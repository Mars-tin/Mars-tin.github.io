Overview

Representing Language Processing

Bag of Tokens

  • Examples:
    • Bag-of-Words (BoW) Models;
    • Topic Modelling.
  • Problems:
    • Ignores the specific positions of tokens in text;
    • Only considers number of occurrences of a unique token in text.

Sequence of Tokens

  • Examples:
    • Linear-chain CRF;
    • Hidden Markov Models (HMM);
    • Sequence Models (RNN, LSTM, Transformers).
  • Problems:
    • Ignores the rich structural information and relationships among textual elements.

Graph

  • Examples:
    • Dependency and Constituency Graphs;
    • Abstract Meaning Representation (AMR) Graphs;
    • Information Extraction (IE) Graphs;
    • Knowledge Graphs (KG);
    • Lexical Networks.

Classic Graph-Based Methods for NLP

Algorithms

  • Random Walk Algorithms
    • Idea: Produce random paths in a graph to converge to a stationary distribution over all the nodes.
    • Purpose:
      • Node saliency
      • Graph similarity
  • Graph Clustering Algorithms
    • Examples:
      • Spectral clustering
      • Random walk clustering
      • Min-cut clustering
    • Purpose:
      • Text clustering
  • Graph Matching Algorithms
    • Examples:
      • Graph Edit Distance
    • Purpose:
      • Graph similarity
      • Text entailment
  • Label Propagation Algorithms (LPA)
    • Idea: Propagate and aggregate labels iteratively across the graph in a semi-supervised manner.
    • Purpose:
      • Word-sense disambiguation
      • Sentiment analysis

Problems

  • Limited Expressive Power
    • Focus on graph structures but ignore the node and edge features/attributes.
  • No Unified and General Framework
    • Different graph-based algorithms have different properties and settings, most of which are task-specific.

Graph Neural Networks

Note that there have been many taxonomies for GNN models. The following fliter-pooling framework was not popular and has been outdated for machine learning community, but is sufficient for NLP context. I’d like to point to a very well-written article Graph Convolutional Networks and Pooling by Jonathan Hui.

Framework

We use AA to denote the adjacency matrix of the graph, h(l)ihi(l) to denote the embedding of vertex vivi at the lthlth GNN layer, and H(l−1)i={h(l−1)1,h(l−1)2,⋯,h(l−1)n}Hi(l−1)={h1(l−1),h2(l−1),⋯,hn(l−1)} to denote the input embeddings at the (l−1)th(l−1)th GNN layer.

GNNs 1) learn node embeddings for each node in the graph and 2) aggregate the node embeddings to produce the graph embeddings.

  • Graph filtering learns new node embeddings given input node embeddings and graph structure, without changing the graph structure;
    h(l)i=ffilter(A,H(l−1))hi(l)=ffilter(A,H(l−1))
  • Graph pooling takes a graph and its node embeddings as inputs and then generates a smaller graph with fewer nodes and its corresponding new node embeddings.
    A′,H′=fpool(A,H)A′,H′=fpool(A,H)

Graph Filtering

  • Spectral-based Graph Filters
    • Based on spectral graph theory, i.e. properties of a graph in relationship to the characteristic polynomial, eigenvalues, and eigenvectors of matrices associated with the graph, such as its adjacency matrix AA or Laplacian matrix LL.
    • Examples:
      • ChebNet

      Convolutional Neural Networks on Graphs with Fast Localized Spectral Filtering
      Michaël Defferrard, Xavier Bresson, Pierre Vandergheynst
      NIPS, 2016.
      [Paper]

      • Graph Convolutional Networks (GCN)

      Semi-Supervised Classification with Graph Convolutional Networks
      Thomas N. Kipf, Max Welling
      ICLR, 2017.
      [Paper]

  • Spatial-based Graph Filters
    • Message passing based on a node’s spatial relations. Models depend on different aggregators to gather information from each node’s neighbors.
    • Examples:
      • Message Passing Neural Network (MPNN)

      Neural Message Passing for Quantum Chemistry
      Justin Gilmer, Samuel S. Schoenholz, Patrick F. Riley, Oriol Vinyals, George E. Dahl
      ICML, 2017.
      [Paper]

      • GraphSage

      Inductive Representation Learning on Large Graphs
      William L. Hamilton, Rex Ying, Jure Leskovec
      NLPS, 2017.
      [Paper]

  • Attention-based Graph Filters
    • Based on the Multi-head Attention mechanism to dynamically learn the attention scores on the edges when performing message passing.
    • Examples:
      • Graph Attention Network (GAT)

      SGraph Attention Networks
      Petar Veličković, Guillem Cucurull, Arantxa Casanova, Adriana Romero, Pietro Liò, Yoshua Bengio
      ICLR, 2018.
      [Paper]

  • Recurrent-based Graph Filters
    • Based on the Gated Recurrent Units (GRU).
    • Examples:
      • Gated Graph Neural Networks (GGNN)

      Gated Graph Sequence Neural Networks
      Yujia Li, Daniel Tarlow, Marc Brockschmidt, Richard Zemel
      ICLR, 2016.
      [Paper]

Graph Pooling

  • Flat Graph Pooling
    • Max-pooling and Average-pooling
    • Max-pooling and Average-pooling following a fully-connected layer
    • BiLSTM aggregation
  • Hierarchical Graph Pooling
    • Subsampling the most important nodes
      • Graph Pooling Layer (gPool)

      Learning Graph Pooling and Hybrid Convolutional Operations for Text Representations
      Hongyang Gao, Yongjun Chen, Shuiwang Ji
      WWW, 2019.
      [Paper]

    • Combines nodes to form supernodes
      • DiffPool

      Hierarchical Graph Representation Learning with Differentiable Pooling
      Rex Ying, Jiaxuan You, Christopher Morris, Xiang Ren, William L. Hamilton, Jure Leskovec
      NIPS, 2018.
      [Paper]

      • EigenPooing

      Graph Convolutional Networks with EigenPooling
      Yao Ma, Suhang Wang, Charu C. Aggarwal, Jiliang Tang
      SIGKDD, 2019.
      [Paper]
